## Requirements

In order to run this project you must have:
- Xcode 10.x
- Swift 4.2 support
- Cocoapods 1.5.x

## Getting Started

To run this project, run the following commands on terminal:
```
git clone https://github.com/matheusvds/Weather-App.git
pod install
```
